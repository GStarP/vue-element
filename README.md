# vue-element

## Introduction

our slogan is `simple`, so we won't add complex feature, such as

- pre-commit hook (husky)
- random mock data (mockjs)

## Scripts

install dependency

```
npm install
```

hot-update develop

```
npm run serve
```

run mock server

```
npm run mock
```

build artifact

```
npm run build
```

lint code (and fix)

```
npm run lint
```
